# Entertainment Tadka Telegram Bot 🎬

A powerful Telegram bot for movie indexing, searching, and requests.

## Features ✨
- Auto-indexing from multiple channels
- Smart movie search
- Request system
- Admin panel
- Multi-channel support (4 public + 2 private)
- JSON database (no CSV)

## Commands 🤖
- `/start` - Welcome message
- `/search [movie]` - Search movies
- `/request [movie]` - Request a movie
- `/myrequests` - View your requests
- `/help` - Help message

### Admin Commands 👑
- `/stats` - Bot statistics
- `/pending` - View pending requests
- `/approve [id]` - Approve request
- `/admin` - Admin panel
